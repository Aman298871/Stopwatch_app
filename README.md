# ⏱ Stopwatch App

A modern and user-friendly Stopwatch application built using **Kotlin** for Android. The app provides accurate time tracking with Start, Stop, Resume, and Restart functionality, wrapped in a clean Material Design interface.

## 📱 Features

* ▶️ Start Timer
* ⏸️ Stop/Pause Timer
* 🔄 Resume Timer
* 🗑️ Restart Timer
* ⏱ Real-time HH:MM:SS Time Format
* 🎨 Modern and Responsive UI
* 🔗 View Binding Integration
* ⚡ Lightweight and Fast

## 🛠️ Tech Stack

* Kotlin
* Android SDK
* View Binding
* Handler & Runnable
* Material Design Components
* CardView

## 📂 Project Structure

* MainActivity.kt
* activity_main.xml
* View Binding
* Drawable Resources
* Material Components

## 🚀 How It Works

1. Tap **Start** to begin the stopwatch.
2. Tap **Stop** to pause the timer.
3. Tap **Resume** to continue from the paused time.
4. Tap **Restart** to reset the stopwatch to 00:00:00.
